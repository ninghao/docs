# 后端应用

你懂了一门后端语言，比如 Ruby，或者 PHP，也明白了怎么用 SQL 从数据库那里得到需要的数据。现在你可以选择学习一个系统或者一个框架。在这里你又会遇到很多分支。如果你从 PHP 那来，我推荐现在学一下 Laravel 框架与 Drupal 系统。如果你是从 Ruby 那儿来，现在可以学一下 Rails 框架。以后你可以基于这些系统与框架去开发自己想要的网站。
